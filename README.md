UWB LOS/NLOS Signal Classification and Distance Estimation1. Project OverviewThis project addresses the challenge of precise indoor localization using Ultra-Wideband (UWB) wireless signals. In indoor environments, obstacles like walls and doors create Non-Line-of-Sight (NLOS) conditions, leading to significant ranging errors.Our solution implements the 3D Process of Data Analytics to:Classify signals as LOS or NLOS for two dominant paths.Estimate the measured range for these paths to improve localization accuracy.2. The 3D Process ImplementationA. Data Preparation (The 1st D)We processed a real empirical dataset containing 42,000 samples across 7 indoor environments.Data Cleaning: Consolidated 7 CSV files and stripped whitespace from headers to ensure consistency.CIR Normalization: Normalized the Channel Impulse Response (CIR) data by dividing absolute values by the RXPACC (Received Preamble Symbols) as per hardware documentation.Feature Extraction: Extracted Path2_Amp and Path2_Delay from raw CIR data to address the "Two Dominant Paths" requirement.Feature Importance: Ranked 15 features using a Random Forest model, identifying RXPACC and CIR_PWR as the primary predictors of NLOS.B. Data Mining (The 2nd D)We implemented a supervised learning pipeline using an 80:20 train-test split.Classification: Compared Random Forest and Multi-Layer Perceptron (MLP) algorithms to identify the LOS/NLOS status.Regression: Developed a Distance Estimator to predict the Measured_range for Path 1 and Path 2.Logic: Adhered to the constraint: If Path 1 is LOS, Path 2 is NLOS. If Path 1 is NLOS, Path 2 is NLOS.C. Data Visualization (The 3rd D)Key performance indicators were plotted to validate the solution:Confusion Matrix: Evaluated classification accuracy (e.g., ~91%).Actual vs. Predicted Scatter Plots: Assessed distance estimation error using RMSE (e.g., ~1.50m for Path 1 and ~1.78m for Path 2).3.
